# 1. Prerequisites

Ubuntu >= 18.04 

ROS >= Melodic

 
# 2. Build
Copy the folder "code_availability/kinematics_model" into your ros workspace.

$ `catkin_make`

$ `source devel/setup.bash`

$ `roslaunch kinematics_model visualize.launch`

$ `rosrun kinematics_model model_visualization`
