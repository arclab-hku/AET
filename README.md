Copy the folder "kinematics_model" into your ros workspace.

---
### 1. Prerequisites

Ubuntu $\ge$ 18.04 

ROS $\ge$ Melodic

---
### 2. Build
`catkin_make`

`source devel/setup.bash`

`roslaunch kinematics_model visualize.launch`

`rosrun kinematics_model model_visualization`
